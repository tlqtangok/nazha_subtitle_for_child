# nazha_subtitle_for_child
draw subtitle from example

## 🔥 哪吒学字大冒险

一款为6岁小朋友设计的学字游戏，基于哪吒动画台词学习汉字。

### 🎮 玩法说明

- **关卡制**：共300关，每一行是一关，数据来自 `gg_learning.txt`
- **朗读阶段**：进入关卡后，系统会朗读句子，同时逐字高亮对应的卡片
- **选字阶段**：朗读完毕后，小朋友需要从卡片中找出目标汉字并点击
- **即时反馈**：选对有正确音效和庆祝动画🎉，选错有提示音效并可继续尝试
- **巩固朗读**：答对后系统自动朗读目标字3遍，边读边高亮卡片，帮助加深记忆
- **自动进关**：朗读3遍结束后自动跳到下一关，无需手动点击
- **自由选关**：可以在关卡选择页面自由选择任意关卡
- **计时挑战**：每关开始计时，记录完成时间

### 📖 使用方法

直接用浏览器打开 `nazha_learning.html` 即可开始游戏，无需安装任何依赖。

### 📝 数据生成

运行 `python3 generate_learning.py` 可从 `nazha_subtitle.txt`（哪吒台词）和 `to_implement_template.txt`（汉字模板）自动生成 `gg_learning.txt`。

每行格式示例：

```
一:孕育了一颗混元珠的一字
```

### ✨ 功能特色

- 🎵 Web Speech API 语音朗读（支持中文）
- 🔊 Web Audio API 音效反馈
- 🌟 进度保存（localStorage）
- 🎨 可爱的渐变色卡片 UI
- ✨ 粒子庆祝动画效果
- 📱 响应式设计，支持手机、平板和PC（一屏内显示所有元素）
- 🔁 答对后自动读字3遍加深记忆，然后自动进入下一关
