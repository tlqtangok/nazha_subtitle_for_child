# nazha_subtitle_for_child
draw subtitle from example

## 🔥 哪吒学字大冒险

一款为6岁小朋友设计的学字游戏，基于哪吒动画台词学习汉字。

### 🎮 玩法说明

- **关卡制**：关卡数量由 `gg_learning.txt` 的行数动态决定，每一行是一关
- **朗读阶段**：进入关卡后，系统会朗读句子，同时逐字高亮对应的卡片
- **选字阶段**：朗读完毕后，小朋友需要从卡片中找出目标汉字并点击
- **即时反馈**：选对有正确音效和庆祝动画🎉，选错有提示音效并可继续尝试
- **巩固朗读**：答对后系统自动朗读目标字3遍（可在设置中关闭），边读边高亮卡片，帮助加深记忆
- **自动进关**：朗读3遍结束后自动跳到下一关，无需手动点击
- **自由选关**：可以在关卡选择页面自由选择任意关卡
- **计时挑战**：每关开始计时，记录完成时间

### 🔵 蓝色标记与复习机制

- 若用户首次点选字就点错了，该关卡在主页上会被标记为**蓝色**（待复习）
- 继续完成 **5 个普通关卡**后，该蓝色关卡会自动插入当前关卡之前，供用户再次复习
- 复习时若仍然首次点错，继续标蓝、再次排入复习队列
- 复习时一次点对，关卡升为**绿色**（已过关）
- 绿色：一次答对，已过关 ✓
- 蓝色：首次答错，等待复习 ↩
- 橙色：尚未完成

### ⚙️ 游戏设置

在主页点击「⚙️ 游戏设置」可展开配置面板：

- **答对后朗读目标字 3 遍**：勾选后答对自动朗读目标字3遍，帮助加深记忆（默认开启）
- **排除关卡**：填入不想出现的关卡序号（逗号分隔，如 `5,10,15`），保存后对应关卡变灰跳过；在主页点击已排除关卡可取消排除

### 📖 使用方法

> **建议通过本地 Web 服务器打开**，因为动态加载 `gg_learning.txt` 需要 HTTP 协议。
>
> 快速启动方法（在项目目录下运行）：
> ```bash
> python3 -m http.server 8080
> # 然后在浏览器访问 http://localhost:8080/nazha_learning.html
> ```

也可直接双击在浏览器打开，部分浏览器（如 Firefox）支持本地文件访问。

### 📝 关卡数据

关卡直接来自 `gg_learning.txt`，**可以手动编辑**该文件来增删或修改关卡。每行一关，格式为：

```
字:包含该字的例句的字字
```

示例：

```
一:孕育了一颗混元珠的一字
二:我要证明我才是十二金仙的最佳人选的二字
```

> `generate_learning.py` 是一个可选的辅助脚本，可从 `nazha_subtitle.txt`（哪吒台词）和 `to_implement_template.txt`（汉字模板）批量生成 `gg_learning.txt`，但无需运行它——直接编辑 `gg_learning.txt` 即可。

### ✨ 功能特色

- 🎵 Web Speech API 语音朗读（支持中文）
- 🔊 Web Audio API 音效反馈
- 🌟 进度保存（localStorage）
- 🎨 可爱的渐变色卡片 UI
- ✨ 粒子庆祝动画效果
- 📱 响应式设计，支持手机、平板和PC（一屏内显示所有元素）
- 🔁 蓝色标记 + 5关后自动复习机制
- ⚙️ 可配置：朗读次数开关、自定义排除关卡
- 📄 关卡数据从 `gg_learning.txt` 动态读取，随文件内容实时更新
