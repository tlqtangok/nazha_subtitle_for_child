# 哪吒学字 - 后台同步功能使用说明

## 新增功能

### 1. 首次使用需要输入令牌（Token）
- 第一次打开页面时，会弹出窗口要求输入令牌
- 输入令牌后，会永久保存在浏览器中，下次打开不需要再输入
- 如果令牌失效，系统会自动清除并要求重新输入

### 2. 标记已学会的字（灰色显示）

#### 如何进入多选模式：
1. 在首页**长按**（按住不放约1秒）任意一个字的按钮
2. 进入多选模式后，页面底部会出现两个按钮：
   - 💾 保存进度
   - ✖ 取消

#### 如何选择字：
- 在多选模式下，**点击**你想要标记为"已学会"的字
- 被选中的字会显示红色边框
- 再次点击可以取消选择
- 顶部会显示"已选择 X 个字"

#### 如何保存：
- 选择完所有已学会的字后，点击底部的 **"💾 保存进度"** 按钮
- 数据会保存到云端服务器
- 保存成功后，被选中的字会变成灰色（30%透明度+灰度滤镜）

#### 如何取消：
- 点击底部的 **"✖ 取消"** 按钮退出多选模式
- 不会保存任何更改

### 3. 查看已学会的字
- 保存后的字会显示为灰色
- 灰色的字表示之前已经标记为"学会"
- 灰色的字仍然可以点击进行学习

### 4. 数据同步
- 所有标记的数据会保存到云端
- 在任何设备上登录（使用相同的令牌），都能看到相同的灰色标记
- 数据会自动在页面加载时从云端获取

## 操作流程示例

```
1. 打开页面
   ↓
2. 输入令牌（仅首次需要）
   ↓
3. 系统自动加载云端数据
   ↓
4. 已学会的字显示为灰色
   ↓
5. 长按任意字进入多选模式
   ↓
6. 点击选择新学会的字
   ↓
7. 点击"保存进度"按钮
   ↓
8. 保存成功，新选的字变灰
```

## 注意事项

- 长按时间约为 0.8 秒，按太短不会触发多选模式
- 手机端支持触摸长按
- 电脑端支持鼠标长按
- 保存时必须至少选择一个字，否则会提示"请先选择要标记为灰色的字"
- 如果网络错误或令牌失效，会显示错误提示

## 数据格式

保存到云端的数据格式为：
```json
{
  "nazha_learning": {
    "七": false,
    "八": false,
    "九": false
  }
}
```

其中 `false` 表示该字已学会，应该显示为灰色。

## 隐私说明

- 令牌存储在浏览器的 localStorage 中，仅存储在本地
- 学习进度数据存储在云端服务器：`https://jesson.tech:10221`
- 数据通过令牌进行验证和隔离，每个用户只能访问自己的数据
